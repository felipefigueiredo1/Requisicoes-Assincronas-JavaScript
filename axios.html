Oque é o AXIOS?

  • É uma biblioteca JavaScript que permite fazer solicitações HTTP facilmente
  • Baseado na FETCH API , mas oferece algumas vantagens adicionais, como a posibilidade de cancelar solicitações e tratar erros de maneira mais fácil
  • Compatível com navegadores modernos e também pode ser usado em aplicativos Node.js
  • Diferente do FETCH não vem por PADRÃO nos navegadores, sendo necessário ser instalado
